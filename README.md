# 2D Platformer
A 2D platformer game developed with Unity and C# as a diploma project. The player controls a character, collects coins, avoids enemies and progresses through levels.
## Features
<img width="1920" height="1080" alt="A screenshot showcasing gameplay" src="https://github.com/user-attachments/assets/c1f71536-2708-4d50-9e2c-31975a692d0e" />

- **Player Movement:** walking, jumping with smooth physics-based controls;
- **Collectibles:** coins for score and shields;
- **Enemies:** moving and unmoving, player loses health on contact;
- **UI:** main menu, health display, score counter.
## Install & Run
### 1st Option: download the latest release (Windows)
1. Go to [Releases](https://github.com/LiliyaEremenko/platformer_/releases)
2. Download platformer.zip
3. Unzip and run the .exe file
### 2nd Option: run from source
1. Clone the repository
```
$ git clone https://github.com/LiliyaEremenko/platformer_.git
$ cd platformer_
```
2. Open the project in Unity Hub
3. Open the Scenes/MainMenu scene and press **Play**
